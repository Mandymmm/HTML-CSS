DOM

获取DOM对象的方法

  • 通过id获取一个元素
    document.getElementById('idName')

    • document是获取范围，通过id获取的时候，范围只能是document
    • 如果页面上有两个以上相同的id名称，获取第一个
    • 在ie67中，会将表单元素的name属性值当做id名称来调取，并且不区分大小写
    • 可以直接调用id代表这个元素（不推荐）

  • 通过标签名获取一组元素
    document.getElementsByTagName('tagName')

    • 获取到的是一个集合，类数组（索引、长度）
    • document是一个可以变的范围
    • 获取的时候通过集合中的索引来进行获取 obj[0] / obj.item(0)

  • 通过标签属性name的值获取一组元素
    document.getElementsByName('name')

    • 获取到的是一个集合，类数组（索引、长度）
    • document是一个可以变的范围
    • 获取的时候通过集合中的索引来进行获取 obj[0] / obj.item(0)
    • 在ie6-9浏览器下，只对表单元素起作用

  • 通过标签属性class的值获取一组元素
    document.getElementsByClassName('className')

    • 获取到的是一个集合，类数组（索引、长度）
    • document是一个可以变的范围
    • 获取的时候通过集合中的索引来进行获取 obj[0] / obj.item(0)
    • 在ie678中报错

  • 通过根元素获取

1.document.documentElement//获取html元素
2.document.body //获取body元素
3.//获取当前页面的宽度和高度
4.var width=document.documentElement.clientWidth||document.body.clientWidth
5.//client [ˈklaɪənt] 客户端 当前 当事人
6.var height=document.documentElement.clientHeight||document.body.clientHeight
  • 在移动端我们获取元素常用的方法（ ie678不兼容）
1.document.querySelector("#id");//获取一个
2.document.querySelectorAll(".class");//获取一组
3.document.querySelectorAll("#tab li");
4.p.s：选择器参考css选择器的规则

DOM节点

Node 在页面中出现的所有东西都是节点，元素、注释、文本等都是节点
应用场景：通过节点可以进行DOM对象的增删改查

常用的DOM节点有哪些？

1.childNodes 获取所有的子节点（一组）
2.children   获取所有元素子节点（一组）
3.parentNode 获取父亲节点
4.previousSibling 获取上一个节点
5.previousElementSibling 上一个元素节点（不兼容）
6.nextSibling 获取弟弟节点   
7.nextElementSibling下一个元素节点（不兼容）
8.firstChild 获取子节点中的第一个
9.firstElementChild （不兼容）
10.lastChild  获取子节点中最后一个
节点类型 nodeType nodeName nodeValue
元素节点（元素标签） 1 大写的标签名 null
文本节点（文字） 3 text 文字内容
注释节点（注释） 8 comment 注释内容
document（整个文档） 9 document null

DOM节点的增删改查

创建元素节点

createElement() create[kriˈet] 创造 创建

将新创建的元素节点添加到指定元素所有内容之后（末尾）

oDiv.parentNode.appendChild(oDiv);

appendChild 前是oDiv的父级元素节点

将创建的元素添加到已有元素之前

父级元素.insertBefore(要放置的元素，放在谁之前)
oldTag.parentNode.insertBefore(newTag,oldTag)

创建的元素于已有元素为兄弟关系

创建属性节点

setAttribute(属性名,属性值)–直接向对象添加一对键值对 attribute[əˈtrɪbjut] 属性
这个属性会在html标签中显示

1.oDiv.setAttribute('zhufeng','peixun');//显示在标签内
2.console.log(oDiv.getAttribute('zhufeng'));
3.oDiv.removeAttribute('zhufeng');

删除创建的元素节点

父级元素.removeChild(删除的对象)
oDiv.parentNode.removeChild(oDiv)
已有的和新创建的元素节点都可以移除

克隆元素节点 [kloʊn]

oDiv.cloneNode(true)
默认参数为false（不填参数）：只克隆当前元素
true ：克隆当前元素与元素的所有子节点

替换元素节点 [rɪˈples]

父级元素.replaceChild(替换的对象，被替换的对象)
oDiv.parentNode.replaceChild(替换的对象，被替换的对象)

Math方法及应用

  • Math.abs() 取绝对值
  • Math.ceil()向上取整 （出现小数点就向上＋1） [si:l]
  • Math.floor()向下取整 [flɔr]
  • Math.round()四舍五入 [raʊnd]
  • Math.max(val1,val2,val3…)取最大值
  • Math.min(val1,val2,val3…)取最小值
  • Math.random()获取[0-1)之间的随机小数（不包含1）[ˈrændəm]
  • Math.round(Math.random()*(m-n)+n) 获取任意两个数之间的随机数
